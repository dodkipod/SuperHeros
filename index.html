Welcome!


This project is about using rest api's and promises/async functions. It is meant to be viewed on a computer and is not responsive, as I didn't want to spend too long on design and layout.

Fill in the inputs below with a number and click a button to see the SuperHeros. Enter a number between 1 and 731 for a successful output (some in the middle dont work either). Enter 0, a number larger than 731 or a letter for a failed output.

X

All at once - All or nothing:

Will wait for all 3 superheros to be ready and then display all 3 superheros at once - IF they are all successful. If one or more fail, no superheros will be displayed.

All at once - Succeed or fail:

Will wait for all 3 superheros to be ready and then display all 3 superheros at once, even if one fails.

Show whatever is ready:

Will display superheros and/or failures as soon as they are ready.

Who's the strongest:

Will calculate which superhero is the strongest out of the three (only activates once superheros are displayed).
